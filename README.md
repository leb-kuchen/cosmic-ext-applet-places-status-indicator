# Install
```sh
git clone https://github.com/leb-kuchen/cosmic-ext-applet-places-status-indicator
cd cosmic-ext-applet-places-status-indicator
cargo b -r
sudo just install
```
# Dependencies
(some may not be required)
```
Build-Depends:
  debhelper (>= 11),
  debhelper-compat (= 11),
  rustc ,
  cargo,
  libdbus-1-dev,
  libegl-dev,
  libpulse-dev,
  libudev-dev,
  libxkbcommon-dev,
  libwayland-dev,

  just,
  pkg-config,
```
